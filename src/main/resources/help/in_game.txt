Available commands
 /h, /help                                                           print help page
 /a, /assistant <id>                                                 play an assistant card
 /sd, /dining <color>                                                move a student to dining room
 /si, /island <id> <color>                                           move a student to an island
 /m, /mother <id>                                                    move Mother Nature to an island
 /cl, /cloud <id>                                                    choose a cloud tile
 /ch, /character <id>                                                play a character card
 /ccargs, /ccarguments [<srccols> <destcols> <targetcol> <island>]   set character card arguments

Utility commands
 /bs, /bstat              print board status
 /es, /estat              print entrance status
 /drs, /dstat             print dining room status
 /cs, /coins              print coins status
 /acl, /acards            print held assistant card list
 /ccl, /ccards            print character card list
 /cc, /cdesc <name>       retrieve a character card description
 /ccs, /ccstat <name>     retrieve a character card status
 /ps, /pstat <username>   retrieve school board status of a player
